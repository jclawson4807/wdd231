Purpose

Once this site is complete and launched publicly, it site will be a resource for people who want to learn about family history, and will include how-to articles on gathering family history related information about your family, how to use online family history tools such as familysearch.org, reviews of online family history sites, and more. The page will include a CTA form which will allow visitors to join our mailing list. It will also include a form which allows users to request research help. Eventually, after the assignment is done, the site will include a moderated community space where people can ask for research help from othe rmembers, as well as basic productivity tools focused on doing family history research.

I have been working on the idea behind this site for some time. I ran a not-for-profit family history research group for many years, and will invite some of the people I worked with on that team to help run this site once it launches. I have previously built two pre-launch versions of this site, and with what I am learning in this course, the site functionality will be substantially improved. I no longer have to use page builder tools. One could say that a purpose of this site, and this assignment, is to help me learn to build something real and useful for other people. I am very grateful for the opportunity to do this, and for what I am learning at BYU Idaho. Thank you.

Scenarios

  • I am new to family history. What is the best way to start learning about my family?
  • How can I start building my family tree?
  • What kind of free resources exist that will help me find my ancestors?

Page Design

The three content screens will all display content using the same section template. The body of an article may contain an image. For the purposes of this assignment, the article data will be included in an array of Javascript objects, which will be loaded from a JSON file on the server. The selection of different screens from the menu will result in the array being filtered by a screen type value. In later iterations of this page, the content will be stored in a database and loaded based on user search queries or by page association.

Page selection information will be stored in local storage, and if the user returns to the page, he or she will be directed to the last page viewed.

The Research Help screen contains a form which allows the user to request help with family history research. After the form is submitted, a form submission response page will be displayed which will summarize the submitted information.

Two modals will be used. One will thank the user for submitting the CTA form. The other will provide information about the author of the articles.

I plan on building a dark mode/light mode switch feature into the page. The display mode status will also be stored in local storage.
